# File Renamer

Небольшая CLI-утилита на Python для безопасного массового переименования файлов в одной папке.

## Что делает

Скрипт принимает путь к папке и подготавливает новые имена файлов по простым правилам:

- заменяет пробелы на `_`
- приводит имена к lower case

По умолчанию утилита работает в режиме preview и только показывает, что будет переименовано.

Реальное переименование выполняется только при передаче флага `--apply`.

## Ограничения первой версии

- работает только с файлами в одной папке
- не обрабатывает подпапки рекурсивно
- сначала показывает план изменений, потом по флагу применяет его

## Планируемый интерфейс

```bash
python renamer.py /path/to/folder
python renamer.py /path/to/folder --apply
